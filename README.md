# 🍎 App de Introdução Alimentar

Um aplicativo focado em auxiliar pais e cuidadores durante a fase de introdução alimentar dos bebês. O app combina curadoria de dados locais com buscas inteligentes na web.

## 🚀 Funcionalidades
- **Busca Inteligente:** Integração com API externa para buscar informações sobre qualquer alimento na internet.
- **Curadoria de Conteúdo:** Filtro automático para transformar resultados técnicos em dicas práticas de introdução alimentar.
- **Diário Alimentar Editável:** Planejador diário com 3 refeições (Café, Almoço e Janta) organizado por cards.
- **Favoritos (Cache Local):** Salvamento de alimentos preferidos usando `localStorage` para acesso rápido offline.
- **Design Responsivo:** Cores suaves (Rosa e Lilás) otimizadas para uso em dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
- **React.js** (Frontend)
- **Tailwind CSS** (Estilização)
- **Lucide React** (Ícones)
- **Vite** (Build Tool)
- **DuckDuckGo API** (Busca Externa)
